# MH3GetSessionErrorCode



Synopsis
--------
```C++
int MH3GetSessionErrorCode(void);
```



Address
-------
 * __RMHJ08:__ 0x803a4768
 * __RMHE08:__ 0x8043500c
 * __RMHP08:__ 0x8042be2c



Description
-----------
Gets the session error code.



Return Value
------------
Returns one of the following error codes:
 * 0
 * 11620~11625
 * 11630~11635
 * 11640~11644
 * 11650~11651
 * 11660~11669
 * 11670~11679
