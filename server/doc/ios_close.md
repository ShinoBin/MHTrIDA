# ios_close



Synopsis
--------
```C++
int ios_close(int fd);
```



Address
-------
 * __RMHJ08:__ 0x80472cb0
 * __RMHE08:__ 0x804bc190
 * __RMHP08:__ 0x804bc810



Description
-----------
Tries to close the device with the specified file descriptor.



Return Value
------------
Return a nonnegative value on success.
